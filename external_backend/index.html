🤝 Preguntas Frecuentes (Frequently Asked Questions)

¡Bienvenido a la sección de Preguntas Frecuentes! Aquí recopilamos las dudas más comunes que puedan surgir con respecto a nuestra plataforma.

🔷 Uso

  • ¿Qué es SmartStock IA? SmartStock IA es un sistema de optimización de inventarios diseñado especialmente para Pymes. Permite transformar datos históricos en decisiones estratégicas de reposición de manera simple y accesible para todo tipo de usuario.
  • ¿Cómo funciona? El sistema analiza el archivo de inventario y entrega una predicción. Solo debes presionar “Cargar CSV o Excel”, subir tu archivo y luego hacer clic en “Ejecutar IA”. Si el archivo presenta algún problema, el chatbot de ayuda se activará automáticamente para guiarte.
  • ¿Qué hace el chatbot? Maneja problemas comunes con los archivos CSV de forma clara y amigable. Indica qué columnas faltan, cuáles presentan errores y brinda ejemplos prácticos para corregirlos.

🔷 Privacidad

  • ¿Hay riesgo de fuga de información? Los archivos se procesan de forma temporal para generar la predicción y luego se descartan. SmartStock IA no almacena los archivos CSV o Excel de sus usuarios.
  • ¿Y las predicciones se guardan? Actualmente, SmartStock IA tampoco guarda las predicciones. Se planea guardar el resultado general de una predicción a futuro y permitir que los usuarios registrados tengan acceso a esta funcionalidad.

🔷 Requisitos técnicos

  • ¿Qué necesito para usar la aplicación? Solo el archivo de inventario, un dispositivo capaz de conectarse a Internet y un navegador.
  • ¿Mi inventario debe cumplir algún requisito? Solo necesita tener las columnas mínimas para funcionar: sku, stock, demand y lead_time.
  • ¿Y qué significan esos conceptos?
    • SKU (Stock Keeping Unit): es un identificador único para cada producto. Permite distinguir artículos en el inventario.
    • Stock: es la cantidad actual disponible de un producto en inventario.
    • Demand: es la demanda estimada o promedio de ventas de un producto en un periodo determinado.
    • Lead_time: es el tiempo que tarda un proveedor en reabastecer un producto desde que se realiza el pedido.
  • ¿Por qué podrían haber problemas con el archivo? El modelo de IA utiliza los nombres de las columnas para identificar la información que necesita para predecir. Si no las identifica, no puede predecir. Es decir, el mapeo de columnas falla. Si el sistema no logra identificar correctamente las columnas necesarias, no podrá generar una predicción válida.
  • ¿No manejaron esto antes? El sistema posee un diccionario de sinónimos como identificación o “mapeo” de columnas primario y una función inteligente para mapeo secundario. Si ambos fallan, aparece el error que activa el chatbot.
  • ¿Qué otros nombres pueden tener las columnas?

    El mapeo primario contempla los siguientes sinónimos:

    • SKU: sku_code, id, producto, item, product_id
    • stock: inventario, cantidad, existencias, qty
    • demand: demanda, consumo, ventas, sales
    • lead_time: leadtime, entrega, plazo_entrega, lead_time_days
    • unit_cost: precio, costo, unit_price, cost

    El mapeo es capaz de comprender el nombre de la columna incluso si las mayúsculas y minúsculas difieren, porque solo necesita que el texto sea igual. Si tanto el mapeo primario y secundario de columnas han fallado, se sugiere revisar el nombre de las columnas y, de ser posible, modificarlo a uno que sea parte de la lista para asegurar su correcta lectura por parte de la aplicación.

  • ¿Es necesario que tenga esas columnas? Sin esas columnas, el sistema no puede generar una predicción, pues la información es insuficiente. Las columnas obligatorias son: SKU, stock, demand, y lead_time. Además de esas 4 columnas obligatorias, el CSV puede incluir las columnas de fecha (o “date”) y unit_cost para mejorar la predicción obtenida.

🔷 Asistencia al usuario

  • ¿Puedo obtener ayuda humana? Lamentablemente, por el momento la asistencia humana no está contemplada como función para el futuro.

SmartStock AI - Código de Ética y Accesibilidad

🧱 1. Código de Ética

SmartStock AI se compromete a desarrollar y operar tecnología de inteligencia artificial con responsabilidad, transparencia y respeto hacia las Pymes y sus comunidades. Nuestro código de ética se basa en cinco principios:

  1. Transparencia en el uso de datos
    • Explicamos claramente qué datos se usan y para qué.
    • No recolectamos información innecesaria.
    • El usuario siempre mantiene el control de sus datos.
  2. Privacidad y protección de la información
    • Los datos de ventas y stock se procesan de forma segura.
    • No compartimos información con terceros sin consentimiento.
    • Aplicamos buenas prácticas de seguridad para evitar filtraciones.
  3. IA responsable y explicable
    • Justificación clara en recomendaciones (demanda, riesgo, nivel óptimo).
    • Evitamos modelos “caja negra” en decisiones críticas.
    • El usuario puede revisar y corregir las recomendaciones.
  4. No discriminación y equidad
    • Sin favoritismos por proveedores, marcas o productos.
    • Decisiones basadas únicamente en datos objetivos.
    • Evitamos sesgos que perjudiquen a negocios pequeños.
  5. Sostenibilidad y eficiencia
    • Reducción de desperdicios mediante inventarios eficientes.
    • Optimización de capital y recursos para Pymes.
    • Fomento de prácticas responsables en la cadena de suministro.

♿ 2. Accesibilidad

Diseñado para todo tipo de usuarios, independientemente de sus capacidades o limitaciones tecnológicas.

  1. Interfaz simple y clara
    • Dashboard intuitivo y lenguaje sin tecnicismos.
    • Recomendaciones directas y accionables.
  2. Compatibilidad técnica
    • Funciona en navegadores estándar y hardware básico.
    • Optimizado para conexiones inestables.
  3. Inclusión y Discapacidad
    • Alto contraste para baja visión y soporte para lectores de pantalla.
    • Botones grandes y etiquetas descriptivas.
  4. Inclusión digital
    • Carga vía CSV simple o Excel para Pymes sin sistemas complejos (ERP).
    • Guías paso a paso integradas.

👥 Sobre Nosotros

Sobre SmartStock IA

Cada día, una Pyme pierde ventas sin notarlo. SmartStock IA nace para cerrar esa brecha, transformando datos históricos en decisiones estratégicas de reposición. No somos solo una herramienta de análisis; somos el motor de acción para el comercio local.

🎯 Nuestra Propuesta

Convertimos la complejidad de la ciencia de datos en recomendaciones claras: cuándo y cuánto reponer. A través de modelos avanzados como Prophet y cálculos de optimización de stock, ayudamos a:

  • Reducir quiebres: No pierdas más ventas por falta de producto.
  • Eliminar inventario muerto: Libera capital inmovilizado en artículos que no rotan.
  • Mejorar el flujo de caja: Compra solo lo que realmente vas a vender.

💡 Tecnología con Propósito

Sabemos que muchas Pymes no cuentan con sistemas ERP complejos. Por eso, diseñamos una interfaz accesible, simple y humana. Nuestra prioridad es eliminar las barreras tecnológicas para que cualquier emprendedor, sin importar su nivel de alfabetización digital, pueda competir en igualdad de condiciones.

🚀 El Futuro

Estamos construyendo el centro de comando definitivo para el retail independiente. Proyectamos integrar alertas automáticas, simuladores de ahorro en tiempo real y conexiones directas para que la gestión inteligente esté al alcance de un solo clic.
